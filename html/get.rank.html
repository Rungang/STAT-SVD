Description

Estimate Tucker rank of a given tensor

Usage

get.rank(Y, sigma, sparse_mode)

Arguments

Y

Input tensor, with dimension p1*p2*...*pd.

sigma

(Estimated) Gaussian noise level

sparse_mode

A d-dimensional logicical vector that indicates whether each mode is sparse.

Value

Estimated rank as a d-dimensinal integer tuple
