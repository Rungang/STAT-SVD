Sparse low-rank tensor decomposition using High-order orthogonal iteration

Description

Sparse low-rank tensor decomposition using High-order orthogonal iteration

Usage

HOOI(Y, r, sparse_mode, tmax, vartol)

Arguments

Y

Input tensor, with dimension p1*p2*...*pd.

r

(Estimated) Tucker ranks.

sparse_mode

A d-dimensional logicical vector that indicates whether each mode is sparse.

tmax

Maximal iteration times.

vartol

Covergence telorance.

Value

Estimated subspace in each mode.
