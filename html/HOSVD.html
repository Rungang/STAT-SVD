Sparse low-rank tensor decomposition using High-order SVD

Description

Sparse low-rank tensor decomposition using High-order SVD

Usage

HOSVD(Y, r, sparse_mode)

Arguments

Y

Input tensor, with dimension p1*p2*...*pd.

r

(Estimated) Tucker ranks.

sparse_mode

A d-dimensional logicical vector that indicates whether each mode is sparse.

Value

Estimated subspace in each mode.
